# **App Name**: Sehat Sathi

## Core Features:

- Rule-Based Chatbot: Provide predefined responses based on user input for health issues, emergencies, and hospital information.
- Health Guidance: Offer basic first aid and health guidance based on common queries like fever and injuries, using a rules engine.
- Hospital Locator: Display a dummy list of nearby hospitals and clinics, including addresses and a Google Maps link. This can leverage tool to make choices about relevance of particular links for the user
- Tele-Consultation Booking Simulation: Simulate the booking of tele-consultations, including name, phone, and issue input fields, followed by a booking confirmation message.
- Multilingual Support: Offer the chatbot and website content in both English and Hindi.
- Contact Form: Implement a contact form for users to submit inquiries, including fields for name, email, and message.
- Homepage Hero Section: Design a hero banner with a compelling title and subtitle, accompanied by a 'Start Chatting' CTA button.

## Style Guidelines:

- Primary color: Moderate cyan (#45B3E0) to convey trustworthiness, aligned with the healthcare context.
- Background color: Light cyan (#E0F7FA), subtly tinted but very desaturated.
- Accent color: Darker blue (#2E86C1), creating emphasis while still staying within the established healthcare context.
- Font: 'PT Sans', a modern, humanist sans-serif font, suitable for both headlines and body text due to its balance of modernity and approachability.
- Use simple, clear icons related to health and assistance to guide users.
- Clean, mobile-friendly layout with rounded buttons and a user-friendly chatbot design.
- Subtle animations for chatbot responses to improve user engagement.
